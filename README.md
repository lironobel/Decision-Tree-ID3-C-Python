# C-Engine Decision Tree Classifier 🌲

A high-performance Decision Tree (ID3/C4.5) engine implemented in C, featuring a modern Python-based visualization dashboard.

## Key Features

- **C-Core:** Custom implementation of decision tree logic from scratch.
- **Memory Efficient:** Manual memory management and optimized recursive structures.
- **Interactive Dashboard:** Built with Streamlit to visualize tree growth and compare results.
- **Accuracy Benchmarking:** Automated comparison against Scikit-Learn models.

## Tech Stack

- **Backend:** C (GCC, Make)
- **Frontend/UI:** Python (Streamlit)
- **Visualization:** Graphviz
- **Version Control:** Git
